Sobre Mim

Olá! Sou um estudante de Desenvolvimento Web apaixonado por transformar ideias em soluções digitais eficientes e dinâmicas. Desde cedo, minha curiosidade por tecnologia me levou à programação, e hoje busco evoluir constantemente na área.

Com forte foco em back-end, minha principal competência é o desenvolvimento de APIs RESTful com C# e .NET, utilizando Entity Framework para a comunicação com bancos de dados. Também possuo experiência prática em PHP, MySQL, Python e, no front-end, utilizo HTML, CSS e JavaScript para criar interfaces intuitivas. Busco minha primeira oportunidade como Estagiário ou Desenvolvedor Júnior para aplicar meus conhecimentos, colaborar com um time e continuar crescendo profissionalmente.
